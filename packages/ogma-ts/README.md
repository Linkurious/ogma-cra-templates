# cra-template-ogma-ts

This is the TypeScript template for [Create React App](https://github.com/facebook/create-react-app) + [Ogma graph visualisation library](https://ogma.linkurio.us).

To use this template, add `--template "@linkurious/ogma-ts"` when creating a new app.

For example:

```sh
npx create-react-app my-app --template "@linkurious/ogma-ts"

# or

yarn create react-app my-app --template "@linkurious/ogma-ts"
```

For more information, please refer to:

- [Getting Started](https://create-react-app.dev/docs/getting-started) – How to create a new app.
- [User Guide](https://create-react-app.dev) – How to develop apps bootstrapped with Create React App.
